# SpaceInvadersGV1

## Development Environment:
* Windows 11 laptop/desktop computer
* 64-bit operating system, x64-based processor
* 32 GB of RAM
* Visual Studio Community 2022
* Windows Forms Application (.NET Framework 4.8)
* C# language version 7.3

## System Requirements:
* Windows 11 computer
* 64-bit operating system, x64-based processor
* At least 18 GB of disk space (due to Visual Studio size)
* At least 16 GB of RAM
* Visual Studio Community 2022 (if 2022 is unavailable, a later edition like 2026 works, too):
    * Workloads:
        * .NET desktop development
        * WinUI application development

## Downloading and Instructions (for use within Visual Studio):
1. Download and install Visual Studio Community 2022 (or higher): https://visualstudio.microsoft.com/downloads/
   * When installing Visual Studio Community, select the following Workloads in the Installer:
      * .NET desktop development
      * WinUI application development
2. Select the green "<> Code" button on this repository's page.
3. Do 1 of the following:
   - A. Manually clone the repository yourself using the web URL: https://github.com/eswaydan/SpaceInvadersGV1.git
   - B. Select "Open with Visual Studio" (will prompt you to clone the repository like in option A)
   - C. Select "Download ZIP" and unzip the downloaded folder in your desired location.
4. After Step 3, either double-click the SpaceInvadersGV1.sln file or right-click and select "Open" to open it in Visual Studio.
5. Select "Release" (from the drop-down where it says "Debug" next to the "Any CPU").
6. Select "Build" from the drop-down menu at the top of Visual Studio and pick "Build Solution" (or hit the F7 key).
7. Select the "Start" button (green filled arrow is next to it) to run the game.
8. After building it in Step 6, you can also open the "obj" folder, open the "Release" folder, and run the SpaceInvadersGV1.exe file from there (by double-clicking on it or right-clicking and picking "Open").

## Downloading and Instructions (for use without Visual Studio):
1. Download the SpaceInvadersGV1.exe file from the Releases.
2. Double-click on the SpaceInvadersGV1.exe file (or right-click and select "Open") to run the game.
